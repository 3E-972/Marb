Charts Module

The Charts module contains a set of viewes compatibles with QAbstractItemModel subclasses.

Each column repsresents a serie of data. Nevertheless, some view, like the simple Pie Chart only use the first column.

The model data are converted in float numbers. The non

The data are converted to floatl numbers. Non-numeric values ​​(alphabetic characters, for example) are considered equal to zero.
The X-axis values ​​are determined by the name of each model row (the value of the corresponding QVerticalHeaderItem). The name of each series is determined by the name of the corresponding column (the value of the corresponding QHorizontalHeaderItem).
