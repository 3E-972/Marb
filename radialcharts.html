Radial Chart

Overview

Radial charts display data as sector on a circle. The radius of a sector indicates the order the value (the highest value has the highest radius).
this view can display several serie of data. A serie of data corresponds to a column in the model.

An example of radial chart:

This view is not as clear and easy to interpret that LinearChart. It simply offers a sexier data representation than LinearChart.

How to use

The RadialChart class can be used as the other QAbstractItemView classes:

RaidalChart chart;
chart.setModel( model );
chart.show();
									
chart = Marb.RaidalChart()
chart.setModel( model )
chart.show()
