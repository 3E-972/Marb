Chart Style

OverView

The ChartStyle class defines the column style in a Linear Chart. Each column has its own style. So, a Linear Chart can be contains bar chart and line chart.

The column style defines:

  • The QPen used to draw the bounds or lines.
  • The QBrush used to fill the shape.
  • The Marb::Type
  • The Marb::Shape used to draw the points.

Types

The types defines the type of chart (line, point, bar, etc.).

Marb::Type has three values:

  • Bar: The value is designed by a rectangle. The rectangle height indicates the value.
  • Point: The values are repsented by dots, squares, triangles, etc.
