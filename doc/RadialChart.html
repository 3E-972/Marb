RadialChart Class

Overview

_calculateOrder( self, value ) [PROTECTED]

Calculates the order of scale for value (i.e. the power of ten greater than value).

_calculateBounds( self ) [PROTECTED]

Calculates the minimum bounds and the maximum bounds (i.e., the minimum and the maximum displayed on the chart). Calculates the order and the tick size(delta between two ticks on the Y axis) of the charts values. If the minimum is equal to the maximum, the minimum bound is equal to minimum - 1 and the maximum bound to maximum + 1

__paintTicks( self, painter ) [PRIVATE]

Paints ticks for axis on the paint device 'painter'.

__init__( self, parent=None ) [PRIVATE]

Constructor. Constructs an instance with the given 'parent'.

calculateLegendRect( self )[PUBLIC]

Calculates the rectangle needed to display the legend according to the title of each column (given by the QHeaderView).

columnStyle( self, column )[PUBLIC]

Returns the style for the column. If no style has been set for the column, returns the style by default

_paintLegend( self, painter ) [PROTECTED]

Paint the legend in the QRect self._legendRect The legend corresponds to the text in the Horizontal QHeaderView and the style defined for each column.

_paintColumnLegend( self, painter, c, pos, metricsH ) [PROTECTED]

Paints the legend for the given column. The kind of legend should be defined by each view (linear and radial represent legend differently). The column legend is represented by a square colored with the pen and brush style and the column name.

defineRects( self )[PUBLIC]

Define the bounding rectangles used to draw the chart. * chartRect: The chart bounds (data + axis) * valueRect: Area where the data representation will be drawn. Should be defined inside chartRect * legendRect: Area where the legend will be drawn * titleRect: Area where the title will be drawn

_setAlphaBeta( self ) [PROTECTED]

Calculates the coefficients alpha and beta used to convert a value in the model in the corresponding value in pixel in the chart. Translates the origin to place the X-Axis.

_calculateOrderedColumn( self ) [PROTECTED]

Returns the order in which the columns will be painted in the viewport. The order is defined according to [ChartStyle::type]. *Bars* will be painted first, then *Lines* and *Points*. *Points* will appear at foreground.

indexAt( self, point )[PUBLIC]

Returns the model index of the item at the viewport coordinates point.

itemPath( self, index )[PUBLIC]

Returns the bounding path for the item pointed by the 'index'.

_scanValues( self ) [PROTECTED]

Scans values in the model to find the minimum and the maximum. Returns the width needed to display the Y scale. If the values are greater than zero, the minimum is equal to 0. If the values are less than 0, the maximum is equal to 0. If a value is not a number (undefined, a string, etc.), she's considered as equal to 0.

save( self, filename, size = None )[PUBLIC]

Save the chart in the given file filename. The file should be an PNG image.

_updateRects( self ) [PROTECTED]

Calculates the size and the position of each view part (legend, title,chart) Called when the view is resized or model has changed.

paintChart( self, painter )[PUBLIC]

Paints the chart on the paint device 'painter'.

_paintValues( self, painter, column ) [PROTECTED]

Paints items on the paint device [painter] for the [column]

itemRect( self, index )[PUBLIC]

Returns the rectangle on the *viewport* occupied by the item at [index] without consider the offset.

process( self )[PUBLIC]

Defines the metrics and components to display the chart. Called when model ha changed.

save( self, filename )[PUBLIC]

Save the render in the file named [filename].

setColumnStyle( self, column, style )[PUBLIC]

Sets the style for the column.

setTitle( self, title )[PUBLIC]

Sets the chart title to title Updates the chart to consider the modification

updateValues( self )[PUBLIC]

Updates every component in the view (item positions, bounding rects, etc.). Called when the model has changed (except if the data has only changed.) or when the view was resized.

valueToPx( self, value )[PUBLIC]

Converts the given value in pixel according to the chart scale. The pixel value is given by the equation: y = alpha * value + beta

visualRegionForSelection( self, selection )[PUBLIC]

Reimplemented to consider that the whole viewport needs to be repainted.
