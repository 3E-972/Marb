Overview

List of available types for the linear chart

darkColorAt( index )[PUBLIC]

Returns the dark color corresponding to the index. Used to set a default color.

lightColorAt( index )[PUBLIC]

Returns the light color corresponding to the index. Used to set a default color.

regularColorAt( index )[PUBLIC]

Returns the regular (not light, not dark) color corresponding to the index. Used to set a default color
