ChartStyle Class

Overview

ChartStyle class contains the options for a render. A ChartStyle is used to custom a serie of data. For example, you can specify the color of the a line and the shape of the points. You should not create a ChartStyle but use the Chart::columnStyle( column ).

__init__( self ) [PRIVATE]

Constructor. Constructs an instance of ChartStyle. You should not construct your own instance. Use Chart::columnStyle() method, instead.

brush( self )[PUBLIC]

returns the brush used to fill some part of the render.

pen( self )[PUBLIC]

Returns the pen used to draw some part of the render (the borders of the bar in a bar chart, for example).

setBrush( self, brush )[PUBLIC]

brush: instance of QBrush Sets the brush used to fill some part of the render (the point shape for a line chart or the bar in a bar chart, for example).

setPen( self, pen )[PUBLIC]

pen: instance of QPen Sets the pen used to draw some part of the render (the borders of the bar in a bar chart, for example).

setShape( self, shape )[PUBLIC]

shape: Instance of Global.Shape Sets the shape used to represent the points in a line chart.

setType( self, chartType )[PUBLIC]

chartType: Instance of Global.Type Sets the render for the serie of data (line, points or bar).

shape( self )[PUBLIC]

Returns the shape used to represent the points in a Point/line chart.

type( self )[PUBLIC]

Sets the render type for the serie of data (line, points or bar).
