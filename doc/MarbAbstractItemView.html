MarbAbstractItemView class

Inherits: QAbstractItemView

Overview

The MarbAstractItemView class provides the basic functionality for Marb item view classes. MarbAstractItemView is the base for every class in Marb. MarbAstractItemView is an abstract class and cannot be instancied. It provides a standard interface to display items from a QAbstractItemModel.

__init__( self, parent=None )[PUBLIC]

Constructor. Constructs an instance with the given 'parent'.

_calculateOrderedColumn( self )[PUBLIC]

Returns the order in which the columns will be painted in the viewport. The order is defined according to ChartStyle::type. The "Bar columns" will be painted first, then the "line column" and the "point column". The "point column" will appear at foreground.
